# Tools 目录

## 当前使用的工具

- `flutter_preview_generic/` — 通用 Flutter 预览模板，供 workflow 的 `buildFlutterPreview` 步骤使用
- `generate_preview_main.py` — 从 `flutter-widgets/*.dart` 自动生成预览入口 `main.dart`
- `phaser3_preview/` — Phaser3 预览相关工具
- `wireframe_preview.html` — 线框图预览 HTML 模板

## 遗留工具

- `_legacy/flutter_preview_social_world/` — `social-world` 项目的专用 Flutter 预览模板（已不再使用，所有项目统一使用 `flutter_preview_generic`）
